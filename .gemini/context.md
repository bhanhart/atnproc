# Generic technical context for the project.

## Programming Language

The project is written in Python.
Use Python version >= 3.9.
Always use proper typing annotations.

## GIT Version Control

Never stage or commit without explicitly asking the user for confirmation.

## Relevant Workspace Directories

All Python source files are located under: '${workspaceFolder}/src/atnproc'

All Configuration files are located under: '${workspaceFolder}/config'

All documentation files are located under: '${workspaceFolder}/docs'

Always ignore files in the following directories under ${workspaceFolder}:

- .venv
- .vscode
- __pycache__
- .git
- unused
- airtel
- example
- log
